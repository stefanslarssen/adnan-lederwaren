# Adnan Lederwaren

Portfolio case: B2B leather goods manufacturer website.

## Quick Start with Docker

```bash
# Clone the repo
git clone https://github.com/stefanslarssen/adnan-lederwaren.git
cd adnan-lederwaren

# Run with Docker Compose
docker-compose up

# Open browser at http://localhost:3003
```

## Alternative: Docker Build

```bash
docker build -t adnan-lederwaren .
docker run -p 3000:80 adnan-lederwaren
```

## Features

- Single-page application (vanilla JS)
- Quote cart system
- Login/Register pages
- Book appointment
- Blog section
- Privacy policy
- Responsive design
